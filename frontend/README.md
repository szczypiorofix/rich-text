# Rich-Tech Editor CMS - frontend layer

## Installation & usage

- Installation:
  - ```npm install```

- run:
  - ```npm run dev```

- Requirements:
  - Node ver 22.12.0
  - For PocketBase a .env file is required:
    - example .env file:
    ```
    VITE_API_URL=http://127.0.0.1:8090
    VITE_APP_ENV=local
    ```
